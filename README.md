# Linux_ip_port_setting
Ubuntu Debaian Centos 網卡與SSH配置腳本
操作方法
<br><br>

# Ubuntu_group_ip
1.ip_range填入IP的主機位置，共有8段  (輸入站群IP表)(gateway)<br>
2.ip_range[0] 與 ip_master  要同一段<br>
3.ip_master填入主IP<br>
4.netmask填入的是 /27<br>
5.填入gateway<br>
6.填入dns<br>
7.用完記得把shell刪掉<br>
8.如果部屬失敗，就要檢查vi /etc/netplan/裡面的資料有沒有正確<br>
<br>
